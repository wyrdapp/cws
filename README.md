# CWS – Kodi Repository

Kodi addon repository. Instalace viz níže.

## Instalace

### 1. Přidej zdroj souborů v Kodi

Nastavení → Správce souborů → Přidat zdroj:

```
https://raw.githubusercontent.com/wyrdapp/cws/main/repo
```

Pojmenuj ho např. `CWS`.

### 2. Nainstaluj repozitář

Doplňky → Instalovat ze souboru ZIP → CWS → `repository.cws/` → `repository.cws-1.0.0.zip`

### 3. Nainstaluj addon

Doplňky → Instalovat z repozitáře → CWS Repository → Video doplňky → Stream Cinema → Nainstalovat
